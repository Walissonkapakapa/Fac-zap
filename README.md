# Fac Zap

**As facs voltaram.**

Um projeto web que recria a vibe das antigas facções de trava zap, mas de forma gamificada e segura dentro do navegador.

### Funcionalidades atuais:
- Interface estilo WhatsApp Web
- Criar Fac
- Grupo da Fac com chat
- Declarar Guerra
- Tudo salvo no navegador

Crie sua fac e comece a guerra!
